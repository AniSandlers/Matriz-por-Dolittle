# Matrz-por-Dolittle
Resuelve un sistema de ecuaciones 4x4 por el método Dolittle
Este código resuelve un sistema de ecuaciones ya definido, puedes cambiarlo, solo debes editar la matriz y el vector soluciones.

matrix =
>[[4, 1, 2, -1],

>[3, -1, 0, 2],

>[2, 2, 4, -1],
 
 >[1, -1, -1, 3]]

solutions =

>[20, 8, 16, 6]
